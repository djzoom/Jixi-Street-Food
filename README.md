## 关键比例（来自标准）

- 冷面面：高筋面粉 : 淀粉 = **10 : 7**；食用碱 : 水 = **7 : 100**；混合粉 : 碱水 = **50 : 21**
- 辣椒油：油 : 辣椒 ≈ **2 : 1**（210～240℃ → 160～180℃，静置 ≥8h）
- 水发辣椒：辣椒面 : 水 : 盐 : 味精 = **250 : 450 : 1 : 2**
- 干豆腐：包心菜 : 干豆腐 = **3 : 10**

> ⚠️ 辣菜的调味、配料、用油量，标准原文一律写「**适量**」、配料只列名称。本仓库辣菜表**只列标准明确内容（用什么料、什么口味、哪种油）+ 上述 ★ 精确比例**，**不编造克数**；冷面计算器中的辣菜调味为「建议起步量」，非标准值。请边做边尝。
